画面入出力

一般にプログラミングでの「画面の入出力」というのはターミナルを用いた標準入力・標準出力を指すと思いますが、
JavaScriptではブラウザ上のDOM操作ができることから、ここではブラウザ画面を用いた入出力について書いていきます。

  • index.html
    •           
            <html>
              <head>
                <title>JavaScriptで画面への入出力テスト</title>
                <link rel="stylesheet" href="style.css">
              </head>
              <body>
                <h1>JavaScriptで画面への入出力テスト</h1>
                
                <!-- 日付入力フォーム -->
                <label for="start">Start Date</label>
                <input type="date" id="start" name="trip-start"
                value="2021-01-19"    <!-- 初期値 -->
                min="2021-01-01"      <!-- 最小値 -->
                max="2021-12-31">  <!-- 最大値 -->

                <!-- テキスト入力フォーム -->
                <label for="name">Name(４〜８文字で入力してください)</label>
                <input type="text" name="name" id="name" required
                minlength="4" <!-- 文字数最小値 -->
                maxlength="8" <!-- 文字数最大値 -->
                size="10">

                <!-- 数値入力フォーム -->
                <label for="number">Number</label>
                <input type="number" name="number" id="number"
                min="0" <!-- 最小値 -->
                max="100"> <!-- 最大値 -->

                <!-- ボタン -->
                <input class="favorite styled"
                      type="button"
                      value="Add to favorites"
                      onclick="getValue()">
                
                <!-- テキスト出力部分 -->
                <p class="outputArea" id="output"></p>
                <script src="input_output.js"></script>
              </body>
            </html>

  • input_output.js
    •           
            // 関数名getValue()：htmlのbuttonに同名のonClickを記述する
            function getValue(){
              var date = document.getElementById('start').value;    // 日付入力フォームの値を取得
              var text = document.getElementById('name').value;     // テキスト入力フォームの値を取得
              var number = document.getElementById('number').value; // 数値入力フォームの値を取得
            
              // デベロッパーツールのconsoleに取得したデータを出力させてみる
              console.log(date + " | " + text + " | " + number);    
              
              // HTMLの出力用のpタグに取得したデータを出力させる
              document.getElementById('output').innerHTML = date + " | " + text + " | " + number;
            }
