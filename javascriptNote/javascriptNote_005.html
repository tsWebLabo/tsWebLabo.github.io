条件分岐

条件分岐とは、プログラム上で指定した条件にあっている場合、ない場合で実行する命令文を変えられる仕組みです。
JavaScriptではif ... elseとswitchの2つの条件文があります。

if...else文

if ... else文では条件がtrueの時に実行する命令文とfalseの時に実行する命令文を分けて書くことができます。

  • 基本構文
    •           
            if( 条件 ) {
              条件がtrueの時の処理
            } else {
              条件がfalseの時の処理
            }
  • 例文1
    •           
            var age = 18;

            if(age >= 20) {
              console.log('成人です');
            } else {
              cosole.log('未成年です');
            }

            // 未成年です（条件分を満たさないためelseのブロックが読まれる ）

            var age = 25;

            if(age >= 20) {
              console.log('成人です');
            } else {
              cosole.log('未成年です');
            }

            // 成人です（条件文を満たしているためtrueである）
  • 例文2
    •           
            var age = 25;

            if(age < 10) {
              console.log('10歳未満です');
            } else if(age < 20) {
              console.log('20歳未満です');
            } else if(age < 30) {
              console.log('30歳未満です');
            } else {
              console.log('30歳以上です');
            }

            // 30歳未満です
switch文

switch文を使うと、プログラムは式を評価し、その式の値をcaseラベルと照合します。
一致すると、プログラムはそのラベルに関連付けられた文を実行します。

  • 構文
    •           
            switch(式) {
              case 条件1:
                条件1を満たす時の処理;
                break;

              case 条件2:
                条件2を満たす時の処理;
                break;

              case 条件3:
                条件3を満たす時の処理;
                break;

              ︙

              default:
                どの条件も満たさない時の処理; 
            }
  • 例文
    • if文での記述
      •             
              var stationery = 'pencil';

              if(stationery === 'pen') {
                console.log('ペンは100円です');
              } else if(stationery === 'eraser') {
                console.log('消しゴムは80円です');
              } else if(stationery === 'ruler') {
                console.log('定規は120円です');
              } else if(stationery === 'pencil') {
                console.log('えんぴつは50円です');
              } else {
                console.log('買いたい文房具を選んでください');
              }
    • switch文での記述
      •             
              var stationery = 'pencil';

              switch(stationery) {
                case 'pen':
                  console.log('ペンは100円です');
                  break;
              
                case 'eraser':
                  console.log('消しゴムは80円です');
                  break;
                
                case 'ruler':
                  console.log('定規は120円です');
                  break;
              
                case 'pencil':
                  console.log('えんぴつは50円です');
                  break;
              
                default:
                  console.log('買いたい文房具を選んでください');
              }
