JavaScript入門！！

JavaScriptを基礎から勉強していきます。

YouTubeライブ配信で勉強した内容をまとめて記録していきます。

JavaScriptとは。

  • クロスプラットフォームで動作する。

    • どのOSでも、どのブラウザでもJavaScriptは使えます。

  • 動的型付け言語

    • 変数のデータ型は宣言が不要（動的型付け、弱い型付け）

  • オブジェクト指向言語

    • オブジェクトのデータ型に区別はない。
    継承はプロトタイプベースの機構を通して行われ、プロパティとメソッドはどんなオブジェクトにも動的に追加できる。

  • スクリプト言語
  • クライアントサイドJavaScript
  • サーバーサイドJavaScript
  • JavaScriptとJavaは別物です。

型

  • 数値
  • 文字列
  • 真偽値
  • Symbol
  • オブジェクト
    • 関数
    • Array(配列)
    • Date
    • RegExp(正規表現)
  • null

    • 意図的に「値がない」ということを示す値

  • undefined

    • 初期化されていない状態を示す。
    値が代入されていない状態を示す値

変数

  • var

    • 最も一般的な宣言キーワード。
    let、constのような制約がない。
    varキーワードで宣言した変数は、変数を宣言した関数ブロックから使用できます。

  • let

    • ブロックレベルの変数を宣言できる。
    宣言されたブロックの外からは変数を使えない。

  • const

    • 変更を意図しない変数を宣言できる。 宣言されたあとに変更ができない。（＝定数）

演算子

  • 代入演算子

    • 代入演算子は右オペランド（演算子を挟んで右側）の値を元に左オペランド（演算子を挟んで左側）に値を代入するもの

    • 代入
      •               
                // xにyを代入する
                var x;
                var y = 10;
                
                x = y;

                // x = 10, y = 10
    • 加算代入
      •               
                // xにx + yを代入する
                var x = 10;
                var y = 10;

                x += y;

                // x = 20, y = 10
    • 減算代入
      •               
                // xにx - yを代入する
                var x = 20;
                var y = 10;

                x -= y;

                // x = 10, y = 10
    • 乗算代入
      •               
                // xにx * yを代入する
                var x = 10;
                var y = 10;

                x *= y;

                // x = 100, y = 10
    • 除算代入
      •               
                // xにx / yを代入する
                var x = 100;
                var y = 10;

                x /= y;

                // x = 10, y = 10
    • 剰余代入
      •               
                // xにx % yを代入する
                var x = 10;
                var y = 10;
                
                x %= y;

                // x = 0, y = 10
    • べき乗代入
      •               
                // xにx ** yを代入する
                var x = 10;
                var y = 10;

                x **= y;

                // x = 10000000000, y= 10
  • 比較演算子
    • 被演算子を比較してその結果が真であるかに基づいて論理値を返す。
    • 被演算子には数値・文字列・論理値・オブジェクトを使用できる。
    • ほとんどの場合、2つの被演算子が異なる型ならばJavaScriptではその被演算子を比較に適した型に変換しようとする。
      一般的に被演算子は数値的に比較される結果となる。
    • 例外として===と!==については等値性を比較する前に被演算子を適合する型に変換しない

    以下に比較演算子の種類と使い方の例をあげていきます。あらかじめ次の通り変数を定義しておきます。 

                  
                var a = 20;
                var b = 10;
                var c = 'Sasaki'

    • 等価(==)
      •               
                a == b;
                // false

                a == c;
                // false
    • 不等価(!=)
      •               
                a != b;
                // true

                a != c;
                // true
    • 厳密等価(===)
      •               
                a === b;
                // false

                a === c;
                // false
    • 厳密不等価(!==)
      •               
                a !== b;
                // true

                a !== c;
                // true
    • より大きい(>)
      •               
                a > b;
                // true

                b > a;
                // false
    • 以上(>=)
      •               
                a >= b;
                // true

                b >= a;
                // false
    • より小さい(<)
      •               
                a < b;
                // false

                b < a;
                // true
    • 以下(<=)
      •               
                a <= b;
                // false

                b <= a;
                // true
  • 算術演算子
    • 被演算子として数値（リテラルまたは変数）をとり１つの数値をかえす。
    • 標準的な算術演算子は、加算(+)・減算(-)・乗算(*)・除算(/)
    • 他のプログラミング言語で「浮動小数点数を用いた場合」と同じ様に機能する
    • 加算
      •               
                12 + 5
                // 17
    • 減算
      •               
                12 - 5
                // 7
    • 乗算
      •               
                12 * 5
                // 60
    • 除算
      •               
                12 / 5
                // 2.4
    • 剰余
      •               
                12 % 5
                // 2
    • インクリメント
      • x++
        •                 
                  var x = 3;
                  x++; // ここでは x = 3
                  x;    // この段階で x = 4 になる
      • ++x
        •                 
                  var x = 3;
                  ++x; // ここで既に x = 4
    • デクリメント
      • x--
        •                 
                  var x = 3;
                  x--; // ここでは x = 3
                  x;   // この段階で x = 2 になる
      • --x
        •                 
                  var x = 3;
                  --x; // この段階で x = 2 になる
    • 単項符号反転

      • 被演算子の符号を反転してその値を返す

                    
                var x = 3;
                -x;

                // -3が返ってくる。
    • 単項プラス

      • 数値でない被演算子の数値への変換を試みる

                    
                +"3";  // 3 (数値型)
                +true  // 1
                +false // 0
    • べき乗演算子
      •               
                2 ** 3;  // 2の3乗 = 8

                10 ** -1 // 10の-1乗 = 0.1
  • 論理演算子

    • 基本的にブール値（Boolean型：true / false）を用いる

    • 論理積 （AND：&&）
      •               
                true && true  // trueを返す
                true && false // falseを返す
                false && true // falseを返す

                false && (3 == 4) // falseを返す

                'Cat' && 'Dog' // "Dog"を返す
                false && 'Cat' // falseを返す
                'Cat' && false // falseを返す
    • 論理和 （OR ：||）
      •               
                true || true   // trueを返す
                false || true  // trueを返す
                true || false  // trueを返す
                false || false // falseを返す

                false || (3 == 4) // falseを返す
                false || (3 == 3) // trueを返す

                'Cat' || 'Dog' // "Cat"を返す
                false || 'Cat' // "Cat"を返す
                'Cat' || false // "Cat"を返す
    • 論理否定（NOT：!）
      •               
                !true  // falseを返す
                !false // trueを返す
                !'Cat' // falseを返す

配列

  • 配列とは
    • リストのようなオブジェクトである
    • 単一オブジェクト内に複数の値をリストとして保持している
    • 配列オブジェクトは変数に格納される
    • リスト内の値に個別にアクセスすることができる
  • 配列をつくる
    • 配列を定義してから要素を代入する方法
      •                 
                  var a = new array(); // 配列aを定義
                  a[0] = 'dog';
                  a[1] = 'cat';
                  a[3] = 'hen';

                  a.length; // 3
    • 配列の定義と要素の代入を同時に行う方法
      •                 
                  var b = ['dog', 'cat', 'hen'];

                  b.length; // 3
  • 配列を呼び出す

    • 配列を呼び出すにはfor文を使って呼び出すことができます。

                
              for(let i = 0; i < a.length; i++) {
                console.log(a[i]);
              }

条件分岐

条件分岐とは、プログラム上で指定した条件にあっている場合、ない場合で実行する命令文を変えられる仕組みです。
JavaScriptではif ... elseとswitchの2つの条件文があります。

if...else文

if ... else文では条件がtrueの時に実行する命令文とfalseの時に実行する命令文を分けて書くことができます。

  • 基本構文
    •             
              if( 条件 ) {
                条件がtrueの時の処理
              } else {
                条件がfalseの時の処理
              }
  • 例文1
    •             
              var age = 18;

              if(age >= 20) {
                console.log('成人です');
              } else {
                cosole.log('未成年です');
              }

              // 未成年です（条件分を満たさないためelseのブロックが読まれる ）

              var age = 25;

              if(age >= 20) {
                console.log('成人です');
              } else {
                cosole.log('未成年です');
              }

              // 成人です（条件文を満たしているためtrueである）
  • 例文2
    •             
              var age = 25;

              if(age < 10) {
                console.log('10歳未満です');
              } else if(age < 20) {
                console.log('20歳未満です');
              } else if(age < 30) {
                console.log('30歳未満です');
              } else {
                console.log('30歳以上です');
              }

              // 30歳未満です
switch文

switch文を使うと、プログラムは式を評価し、その式の値をcaseラベルと照合します。
一致すると、プログラムはそのラベルに関連付けられた文を実行します。

  • 構文
    •             
              switch(式) {
                case 条件1:
                  条件1を満たす時の処理;
                  break;

                case 条件2:
                  条件2を満たす時の処理;
                  break;

                case 条件3:
                  条件3を満たす時の処理;
                  break;

                ︙

                default:
                  どの条件も満たさない時の処理; 
              }
  • 例文
    • if文での記述
      •               
                var stationery = 'pencil';

                if(stationery === 'pen') {
                  console.log('ペンは100円です');
                } else if(stationery === 'eraser') {
                  console.log('消しゴムは80円です');
                } else if(stationery === 'ruler') {
                  console.log('定規は120円です');
                } else if(stationery === 'pencil') {
                  console.log('えんぴつは50円です');
                } else {
                  console.log('買いたい文房具を選んでください');
                }
    • switch文での記述
      •               
                var stationery = 'pencil';

                switch(stationery) {
                  case 'pen':
                    console.log('ペンは100円です');
                    break;
                
                  case 'eraser':
                    console.log('消しゴムは80円です');
                    break;
                  
                  case 'ruler':
                    console.log('定規は120円です');
                    break;
                
                  case 'pencil':
                    console.log('えんぴつは50円です');
                    break;
                
                  default:
                    console.log('買いたい文房具を選んでください');
                }

for文

for文は条件式がfalseと評価されるまでループします。

基本構文で見るfor文の処理の流れ

  1. 初期化式を実行
  2. 条件式の評価
    • trueの場合 ：ループ文が実行される
    • falseの場合：forループ文が終了する
  3. 加算式を実行
  4. 2. に戻る
  • 基本構文
    •             
              for([初期化式]; [条件式]; [加算式]) {
                命令文;
              }
  • 例1 Hello World!!を5回繰り返す
    •             
              for(let i = 0; i < 5; i++) {
                console.log('Hello World!!');
              }

              Hello World!!
              Hello World!!
              Hello World!!
              Hello World!!
              Hello World!!
  • 例2 無限ループ：Hello World!!がとまらない
    •             
              for(let i = 0; i >= 0; i++) {
                console.log('Hello World!!');
              }

              Hello World!!
              Hello World!!
              Hello World!!
              ︙
  • 例3 for文を使って配列の要素を全部表示させる
    •             
              var a = ['みかん', 'りんご', 'いちご', 'すいか', 'ぶどう', 'ゆず'];

              for(let i = 0; i < a.length; i++) {
                console.log(a[i]);
              }

              みかん
              りんご
              いちご
              すいか
              ぶどう
              ゆず
do...while文

基本構文にあるとおりdo...while文は実行される命令文が条件式より前に記述される。
do...while文では条件式の判定の前に一度do{}の中の命令文が実行される。
その後条件式の判定がされ、条件式がfalseになる直前まで繰り返される。

  • 基本構文
    •             
              do {
                文;
              } while(条件式);
  • 例文
    •             
              let i = 0;

              do {
                i += 1;
                console.log(i);
              } while(i < 5);

              1
              2
              3
              4
              5
while文

条件式の判定を行い条件式＝falseになるまで命令文を繰り返す

  • 基本構文
    •             
              while(条件式) {
                命令文
              }
  • 例文1
    •             
              let n = 0;
              let x = 0;

              while(n < 3) {
                n++;
                x += n;

                console.log("n=" + n " | x=" + x);
              }

              n=1 | x=1
              n=2 | x=3
              n=3 | x=6
  • 例文2：無限ループ

    • 条件式がfalseにならない場合、無限ループになる。
    プログラムをしっかり設計し、条件式がfalseになることを確認し無限ループを避けましょう。 

                
              while(true) {
                cosole.log('Hello World!!');
              }

              Hello World!!

              Hello World!!
