JavaScript入門！！

JavaScriptを基礎から勉強していきます。

YouTubeライブ配信で勉強した内容をまとめて記録していきます。

JavaScriptとは。

  • クロスプラットフォームで動作する。

    • どのOSでも、どのブラウザでもJavaScriptは使えます。

  • 動的型付け言語

    • 変数のデータ型は宣言が不要（動的型付け、弱い型付け）

  • オブジェクト指向言語

    • オブジェクトのデータ型に区別はない。
    継承はプロトタイプベースの機構を通して行われ、プロパティとメソッドはどんなオブジェクトにも動的に追加できる。

  • スクリプト言語
  • クライアントサイドJavaScript
  • サーバーサイドJavaScript
  • JavaScriptとJavaは別物です。

型

  • 数値
  • 文字列
  • 真偽値
  • Symbol
  • オブジェクト
    • 関数
    • Array(配列)
    • Date
    • RegExp(正規表現)
  • null

    • 意図的に「値がない」ということを示す値

  • undefined

    • 初期化されていない状態を示す。
    値が代入されていない状態を示す値

変数

  • var

    • 最も一般的な宣言キーワード。
    let、constのような制約がない。
    varキーワードで宣言した変数は、変数を宣言した関数ブロックから使用できます。

  • let

    • ブロックレベルの変数を宣言できる。
    宣言されたブロックの外からは変数を使えない。

  • const

    • 変更を意図しない変数を宣言できる。 宣言されたあとに変更ができない。（＝定数）

演算子

  • 代入演算子

    • 代入演算子は右オペランド（演算子を挟んで右側）の値を元に左オペランド（演算子を挟んで左側）に値を代入するもの

    • 代入
      •               
                // xにyを代入する
                var x;
                var y = 10;
                
                x = y;

                // x = 10, y = 10
    • 加算代入
      •               
                // xにx + yを代入する
                var x = 10;
                var y = 10;

                x += y;

                // x = 20, y = 10
    • 減算代入
      •               
                // xにx - yを代入する
                var x = 20;
                var y = 10;

                x -= y;

                // x = 10, y = 10
    • 乗算代入
      •               
                // xにx * yを代入する
                var x = 10;
                var y = 10;

                x *= y;

                // x = 100, y = 10
    • 除算代入
      •               
                // xにx / yを代入する
                var x = 100;
                var y = 10;

                x /= y;

                // x = 10, y = 10
    • 剰余代入
      •               
                // xにx % yを代入する
                var x = 10;
                var y = 10;
                
                x %= y;

                // x = 0, y = 10
    • べき乗代入
      •               
                // xにx ** yを代入する
                var x = 10;
                var y = 10;

                x **= y;

                // x = 10000000000, y= 10
  • 比較演算子
    • 被演算子を比較してその結果が真であるかに基づいて論理値を返す。
    • 被演算子には数値・文字列・論理値・オブジェクトを使用できる。
    • ほとんどの場合、2つの被演算子が異なる型ならばJavaScriptではその被演算子を比較に適した型に変換しようとする。
      一般的に被演算子は数値的に比較される結果となる。
    • 例外として===と!==については等値性を比較する前に被演算子を適合する型に変換しない

    以下に比較演算子の種類と使い方の例をあげていきます。あらかじめ次の通り変数を定義しておきます。 

                  
                var a = 20;
                var b = 10;
                var c = 'Sasaki'

    • 等価(==)
      •               
                a == b;
                // false

                a == c;
                // false
    • 不等価(!=)
      •               
                a != b;
                // true

                a != c;
                // true
    • 厳密等価(===)
      •               
                a === b;
                // false

                a === c;
                // false
    • 厳密不等価(!==)
      •               
                a !== b;
                // true

                a !== c;
                // true
    • より大きい(>)
      •               
                a > b;
                // true

                b > a;
                // false
    • 以上(>=)
      •               
                a >= b;
                // true

                b >= a;
                // false
    • より小さい(<)
      •               
                a < b;
                // false

                b < a;
                // true
    • 以下(<=)
      •               
                a <= b;
                // false

                b <= a;
                // true
  • 算術演算子
    • 被演算子として数値（リテラルまたは変数）をとり１つの数値をかえす。
    • 標準的な算術演算子は、加算(+)・減算(-)・乗算(*)・除算(/)
    • 他のプログラミング言語で「浮動小数点数を用いた場合」と同じ様に機能する
    • 加算
      •               
                12 + 5
                // 17
    • 減算
      •               
                12 - 5
                // 7
    • 乗算
      •               
                12 * 5
                // 60
    • 除算
      •               
                12 / 5
                // 2.4
    • 剰余
      •               
                12 % 5
                // 2
    • インクリメント
      • x++
        •                 
                  var x = 3;
                  x++; // ここでは x = 3
                  x;    // この段階で x = 4 になる
      • ++x
        •                 
                  var x = 3;
                  ++x; // ここで既に x = 4
    • デクリメント
      • x--
        •                 
                  var x = 3;
                  x--; // ここでは x = 3
                  x;   // この段階で x = 2 になる
      • --x
        •                 
                  var x = 3;
                  --x; // この段階で x = 2 になる
    • 単項符号反転

      • 被演算子の符号を反転してその値を返す

                    
                var x = 3;
                -x;

                // -3が返ってくる。
    • 単項プラス

      • 数値でない被演算子の数値への変換を試みる

                    
                +"3";  // 3 (数値型)
                +true  // 1
                +false // 0
    • べき乗演算子
      •               
                2 ** 3;  // 2の3乗 = 8

                10 ** -1 // 10の-1乗 = 0.1
  • 論理演算子

    • 基本的にブール値（Boolean型：true / false）を用いる

    • 論理積 （AND：&&）
      •               
                true && true  // trueを返す
                true && false // falseを返す
                false && true // falseを返す

                false && (3 == 4) // falseを返す

                'Cat' && 'Dog' // "Dog"を返す
                false && 'Cat' // falseを返す
                'Cat' && false // falseを返す
    • 論理和 （OR ：||）
      •               
                true || true   // trueを返す
                false || true  // trueを返す
                true || false  // trueを返す
                false || false // falseを返す

                false || (3 == 4) // falseを返す
                false || (3 == 3) // trueを返す

                'Cat' || 'Dog' // "Cat"を返す
                false || 'Cat' // "Cat"を返す
                'Cat' || false // "Cat"を返す
    • 論理否定（NOT：!）
      •               
                !true  // falseを返す
                !false // trueを返す
                !'Cat' // falseを返す

配列

  • 配列とは
    • リストのようなオブジェクトである
    • 単一オブジェクト内に複数の値をリストとして保持している
    • 配列オブジェクトは変数に格納される
    • リスト内の値に個別にアクセスすることができる
  • 配列をつくる
    • 配列を定義してから要素を代入する方法
      •                 
                  var a = new array(); // 配列aを定義
                  a[0] = 'dog';
                  a[1] = 'cat';
                  a[3] = 'hen';

                  a.length; // 3
    • 配列の定義と要素の代入を同時に行う方法
      •                 
                  var b = ['dog', 'cat', 'hen'];

                  b.length; // 3
  • 配列を呼び出す

    • 配列を呼び出すにはfor文を使って呼び出すことができます。

                
              for(let i = 0; i < a.length; i++) {
                console.log(a[i]);
              }

              "dog"
              "cat"
              "hen"
  • 項目の追加と削除
    •             
              // あらかじめ配列を定義しておきます
              let myArray = ['札幌', '仙台', '東京', '名古屋', '大阪', '博多'];
            
          
                
              // 配列の呼び出し
              for(let i = 0; i < myArray.length; i++) {
                console.log(myArray[i]);
              }

              札幌
              仙台
              東京
              名古屋
              大阪
              博多
    • 項目の追加

      • 配列の最後に項目を追加するにはpush()を使います。
      push()を使う時はひとつ以上の項目を引数に指定します。 

                    
                myArray.push('横浜');

                myArray.push('神戸', '広島', '京都');
              
            
                    
                // 配列の呼び出し
                for(let i = 0; i < myArray.length; i++) {
                  console.log(myArray[i]);
                }
  
                札幌
                仙台
                東京
                名古屋
                大阪
                博多
                横浜
                神戸
                広島
                京都
    • 項目の削除

      • 配列の最後の項目を削除するにはpop()を使います。 

                    
                myArray.pop();
              
            
                    
                // 配列の呼び出し
                for(let i = 0; i < myArray.length; i++) {
                  console.log(myArray[i]);
                }
