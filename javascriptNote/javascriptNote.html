JavaScriptとは。

  • クロスプラットフォームで動作する。

    • どのOSでも、どのブラウザでもJavaScriptは使えます。

  • 動的型付け言語

    • 変数のデータ型は宣言が不要（動的型付け、弱い型付け）

  • オブジェクト指向言語

    • オブジェクトのデータ型に区別はない。
    継承はプロトタイプベースの機構を通して行われ、プロパティとメソッドはどんなオブジェクトにも動的に追加できる。

  • スクリプト言語
  • クライアントサイドJavaScript
  • サーバーサイドJavaScript
  • JavaScriptとJavaは別物です。

型

  • 数値
  • 文字列
  • 真偽値
  • Symbol
  • オブジェクト
    • 関数
    • Array(配列)
    • Date
    • RegExp(正規表現)
  • null

    • 意図的に「値がない」ということを示す値

  • undefined

    • 初期化されていない状態を示す。
    値が代入されていない状態を示す値

変数

  • var

    • 最も一般的な宣言キーワード。
    let、constのような制約がない。
    varキーワードで宣言した変数は、変数を宣言した関数ブロックから使用できます。

  • let

    • ブロックレベルの変数を宣言できる。
    宣言されたブロックの外からは変数を使えない。

  • const

    • 変更を意図しない変数を宣言できる。 宣言されたあとに変更ができない。（＝定数）

条件分岐

条件分岐とは、プログラム上で指定した条件にあっている場合、ない場合で実行する命令文を変えられる仕組みです。
JavaScriptではif ... elseとswitchの2つの条件文があります。

if...else文

if ... else文では条件がtrueの時に実行する命令文とfalseの時に実行する命令文を分けて書くことができます。

  • 基本構文
    •             
              if( 条件 ) {
                条件がtrueの時の処理
              } else {
                条件がfalseの時の処理
              }
  • 例文1
    •             
              var age = 18;

              if(age >= 20) {
                console.log('成人です');
              } else {
                cosole.log('未成年です');
              }

              // 未成年です（条件分を満たさないためelseのブロックが読まれる ）

              var age = 25;

              if(age >= 20) {
                console.log('成人です');
              } else {
                cosole.log('未成年です');
              }

              // 成人です（条件文を満たしているためtrueである）
  • 例文2
    •             
              var age = 25;

              if(age < 10) {
                console.log('10歳未満です');
              } else if(age < 20) {
                console.log('20歳未満です');
              } else if(age < 30) {
                console.log('30歳未満です');
              } else {
                console.log('30歳以上です');
              }

              // 30歳未満です
switch文

switch文を使うと、プログラムは式を評価し、その式の値をcaseラベルと照合します。
一致すると、プログラムはそのラベルに関連付けられた文を実行します。

  • 構文
    •             
              switch(式) {
                case 条件1:
                  条件1を満たす時の処理;
                  break;

                case 条件2:
                  条件2を満たす時の処理;
                  break;

                case 条件3:
                  条件3を満たす時の処理;
                  break;

                ︙

                default:
                  どの条件も満たさない時の処理; 
              }

for文

for文は条件式がfalseと評価されるまでループします。

基本構文で見るfor文の処理の流れ

  1. 初期化式を実行
  2. 条件式の評価
    • trueの場合 ：ループ文が実行される
    • falseの場合：forループ文が終了する
  3. 加算式を実行
  4. 2. に戻る

              for([初期化式]; [条件式]; [加算式]) {
                命令文;
              }

do...while文

その後条件式の判定がされ、条件式がfalseになる直前まで繰り返される。

関数

関数とは
  • 関数は一連の処理の集まりである
  • 関数は外部（再帰の場合は内部）から呼ばれることのできる「サブプログラム」
  • 値を関数に渡すことができ（＝引数）、関数は値を返すことができる（＝戻り値）
  • 関数はFunctionオブジェクトである
関数を定義する
  • 関数の宣言
    • 基本構文
      •               
                function 関数名(引数1, 引数2, ...) {
                  関数の本体;
                }
    • 例文：四則演算の関数を書いてみる
      •               
                // 足し算の関数の定義
                function add(number01, number02) {
                  return number01 + number02;
                }

                // 引き算の関数の定義
                function sub(number01, number02) {
                  return number01 - number02;
                }

                // 掛け算の関数の定義
                function multiple(number01, number02) {
                  return number01 * number02; 
                }

                // 割り算の関数の定義
                function division(number01, number02) {
                  return number01 / number02;
                }

                // 剰余の関数の定義
                function surplus (number01, number02) {
                  return number01 % number02;
                }

                // ２つの値（数字）を変数に格納
                var num01 = 15;
                var num02 = 10;

                // 関数の呼び出し
                var answer01 = add(num01, num02);
                var answer02 = sub(num01, num02);
                var answer03 = multiple(num01, num02);
                var answer04 = division(num01, num02);
                var answer05 = surplus(num01, num02);

                // ターミナルへの出力
                console.log('足し算の結果は ' + answer01);
                console.log('引き算の結果は ' + answer02);
                console.log('掛け算の結果は ' + answer03);
                console.log('割り算の結果は ' + Math.floor(answer04) + ' 余り ' + answer05);

                // 出力結果
                足し算の結果は 25
                引き算の結果は 5
                掛け算の結果は 150
                割り算の結果は 1 余り 5
  • 関数式
    • 基本構文
      •

                }

関数の呼び出し
  • 関数を定義しても、その関数が実行されるわけではない
  • 関数は、指定した一連の処理に名前を付けたもの
  • 関数に引数をわたし呼び出すことで、指定された処理を実行し結果を返してくれる。

関数の呼び出し方法

          
            // 関数の定義：定義されただけ
            function hello(hello) {
              console.log(hello);
            }

            // 関数の呼び出し：ここで関数が実行される
            hello('Good morning!!');
            hello('Good afternoon!!');
            hello('Good evening!!');

            // 結果
            Good morning!!
            Good afternoon!!
            Good evening!!
関数のスコープ

  • スコープ＝定義されたもの（変数・関数）を参照できる範囲
  • スコープには、グローバルスコープとローカルスコープがある
  • 関数の内部で宣言された変数は関数の外部からアクセスできない
  • 関数は自身が定義されたスコープ内で定義されている全ての変数や関数にアクセスできる

          
            // 関数のスコープ
            // グローバルスコープで変数を定義
            var num1 = 20;
            var num2 = 3;
            var name = 'Sasaki';

            // グローバルスコープで関数を定義
            function multiply() {
              return num1 * num2;
            }

            console.log(multiply());

            // 結果
            60

            // グローバル ↔ ローカル

            function getScore() {
              var num1 = 2;
              var num2 = 3;

              function add() {
                return name + ' scored ' + (num1 + num2);
              }
              return add();
            }

            console.log(getScore());

            // 結果
            Sasaki scored 5

上記環境で僕が実際にnode.jsを導入した方法を記載します。

    2. まずは下記の3つのコマンドを入力してnode.jsとnpm(Node Package Manager)をインストールしましょう。

    3.

              sudo ln /usr/local/bin/node /usr/bin/node
  3. 最初にインストールしたnode.js npmは使わないので、アンインストールしておく

    4. 最初にインストールされたnode.jsはバージョン指定したものではなく、その時に選ばれた（おそらく最新のもの）がインストールされているためアンインストールします。

node.jsやSQLite3のデータベースファイルを格納しておくディレクトリ（フォルダ）をつくります。
ディレクトリの場所はお好みの場所で構いません。

データベースアクセスのコードを書こう！！
            
              // SQLite3を使う準備
              var sqlite3 = require('sqlite3').verbose();
              // DBファイルを指定
              var db = new sqlite3.Database('test.db');

              // DB操作
              db.serialize(() => {
                // テーブルを初期化
                dataInitialize();

                // データを格納
                insertData('Sasaki');
                insertData('Satou');
                insertData('Takahashi');

                // 格納したデータを抽出
                selectAll();

                // データを更新
                updateData('Takahashi', 'Tanaka');

                // 格納したデータを抽出
                selectAll();

                // データを削除
                deleteData('Tanaka');

                // 格納したデータを抽出
                selectAll();
              });
              db.close();


              /**
              * テーブルの初期化メソッド
              */
              function dataInitialize() {
                // テーブルを削除
                db.run('DROP TABLE testTable');

                // テーブルを定義
                db.run("CREATE TABLE testTable (id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT)");
                console.log("テーブルの初期化完了");
              }

              /**
              * SELECT文：DBのデータを抽出メソッド
              */
              function selectAll() {
                db.each("SELECT * FROM testTable", (error, row) => {
                  if(error) {
                    console.error("Error", error);
                    return;
                  }
                  console.log(row.id + ' | ' + row.name);
                });
              }

              /**
              * INSERT文：データを新規に格納するメソッド
              * @param {新規追加の名前} name 
              */
              function insertData(name) {
                const stmt_insert = db.prepare("INSERT INTO testTable (name) VALUES (?)");
                stmt_insert.run(name);
                stmt_insert.finalize();
                console.log("データの格納完了");
              }

              /**
              * 既存のデータを更新するメソッド
              * @param {変更前の名前} oldName 
              * @param {変更後の名前} newName 
              */
              function updateData(oldName, newName) {
                const stmt_update = db.prepare("UPDATE testTable SET name = ? WHERE name = ?");
                stmt_update.run([newName, oldName]);
                stmt_update.finalize();
                console.log("データの更新完了");
              }

              /**
              * データを削除するメソッド
              * @param {削除する名前} name 
              */
              function deleteData(name) {
                const stmt_delete = db.prepare("DELETE FROM testTable WHERE name = ?");
                stmt_delete.run(name);
                stmt_delete.finalize();
                console.log("データの削除完了");
              }
