JavaScript入門！！

JavaScriptを基礎から勉強していきます。

YouTubeライブ配信で勉強した内容をまとめて記録していきます。

JavaScriptとは。

  • クロスプラットフォームで動作する。

    • どのOSでも、どのブラウザでもJavaScriptは使えます。

  • 動的型付け言語

    • 変数のデータ型は宣言が不要（動的型付け、弱い型付け）

  • オブジェクト指向言語

    • オブジェクトのデータ型に区別はない。
    継承はプロトタイプベースの機構を通して行われ、プロパティとメソッドはどんなオブジェクトにも動的に追加できる。

  • スクリプト言語
  • クライアントサイドJavaScript
  • サーバーサイドJavaScript
  • JavaScriptとJavaは別物です。

型

  • 数値
  • 文字列
  • 真偽値
  • Symbol
  • オブジェクト
    • 関数
    • Array(配列)
    • Date
    • RegExp(正規表現)
  • null

    • 意図的に「値がない」ということを示す値

  • undefined

    • 初期化されていない状態を示す。
    値が代入されていない状態を示す値

変数

  • var

    • 最も一般的な宣言キーワード。
    let、constのような制約がない。
    varキーワードで宣言した変数は、変数を宣言した関数ブロックから使用できます。

  • let

    • ブロックレベルの変数を宣言できる。
    宣言されたブロックの外からは変数を使えない。

  • const

    • 変更を意図しない変数を宣言できる。 宣言されたあとに変更ができない。（＝定数）

配列

  • 配列とは
    • リストのようなオブジェクトである
    • 単一オブジェクト内に複数の値をリストとして保持している
    • 配列オブジェクトは変数に格納される
    • リスト内の値に個別にアクセスすることができる
  • 配列をつくる
    • 配列を定義してから要素を代入する方法
      •                 
                  var a = new array(); // 配列aを定義
                  a[0] = 'dog';
                  a[1] = 'cat';
                  a[3] = 'hen';

                  a.length; // 3
    • 配列の定義と要素の代入を同時に行う方法
      •                 
                  var b = ['dog', 'cat', 'hen'];

                  b.length; // 3
  • 配列を呼び出す

    • 配列を呼び出すにはfor文を使って呼び出すことができます。

                
              for(let i = 0; i < a.length; i++) {
                console.log(a[i]);
              }

              "dog"
              "cat"
              "hen"
  • 項目の追加と削除
    •             
              // あらかじめ配列を定義しておきます
              let myArray = ['札幌', '仙台', '東京', '名古屋', '大阪', '博多'];
            
          
                
              // 配列の呼び出し
              for(let i = 0; i < myArray.length; i++) {

      • 配列の最後に項目を追加するにはpush()を使います。
      push()を使う時はひとつ以上の項目を引数に指定します。 

                    
                myArray.push('横浜');

                myArray.push('神戸', '広島', '京都');
              
            
                    
                // 配列の呼び出し

      • 配列の最後に項目を削除するにはpop()を使います。
