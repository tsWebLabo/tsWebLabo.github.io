演算子

  • 代入演算子

    • 代入演算子は右オペランド（演算子を挟んで右側）の値を元に左オペランド（演算子を挟んで左側）に値を代入するもの

    • 代入
      •             
              // xにyを代入する
              var x;
              var y = 10;
              
              x = y;

              // x = 10, y = 10
    • 加算代入
      •             
              // xにx + yを代入する
              var x = 10;
              var y = 10;

              x += y;

              // x = 20, y = 10
    • 減算代入
      •             
              // xにx - yを代入する
              var x = 20;
              var y = 10;

              x -= y;

              // x = 10, y = 10
    • 乗算代入
      •             
              // xにx * yを代入する
              var x = 10;
              var y = 10;

              x *= y;

              // x = 100, y = 10
    • 除算代入
      •             
              // xにx / yを代入する
              var x = 100;
              var y = 10;

              x /= y;

              // x = 10, y = 10
    • 剰余代入
      •             
              // xにx % yを代入する
              var x = 10;
              var y = 10;
              
              x %= y;

              // x = 0, y = 10
    • べき乗代入
      •             
              // xにx ** yを代入する
              var x = 10;
              var y = 10;

              x **= y;

              // x = 10000000000, y= 10
  • 比較演算子
    • 被演算子を比較してその結果が真であるかに基づいて論理値を返す。
    • 被演算子には数値・文字列・論理値・オブジェクトを使用できる。
    • ほとんどの場合、2つの被演算子が異なる型ならばJavaScriptではその被演算子を比較に適した型に変換しようとする。
      一般的に被演算子は数値的に比較される結果となる。
    • 例外として===と!==については等値性を比較する前に被演算子を適合する型に変換しない

    以下に比較演算子の種類と使い方の例をあげていきます。あらかじめ次の通り変数を定義しておきます。 

                
              var a = 20;
              var b = 10;
              var c = 'Sasaki'

    • 等価(==)
      •             
              a == b;
              // false

              a == c;
              // false
    • 不等価(!=)
      •             
              a != b;
              // true

              a != c;
              // true
    • 厳密等価(===)
      •             
              a === b;
              // false

              a === c;
              // false
    • 厳密不等価(!==)
      •             
              a !== b;
              // true

              a !== c;
              // true
    • より大きい(>)
      •             
              a > b;
              // true

              b > a;
              // false
    • 以上(>=)
      •             
              a >= b;
              // true

              b >= a;
              // false
    • より小さい(<)
      •             
              a < b;
              // false

              b < a;
              // true
    • 以下(<=)
      •             
              a <= b;
              // false

              b <= a;
              // true
  • 算術演算子
    • 被演算子として数値（リテラルまたは変数）をとり１つの数値をかえす。
    • 標準的な算術演算子は、加算(+)・減算(-)・乗算(*)・除算(/)
    • 他のプログラミング言語で「浮動小数点数を用いた場合」と同じ様に機能する
    • 加算
      •             
              12 + 5
              // 17
    • 減算
      •             
              12 - 5
              // 7
    • 乗算
      •             
              12 * 5
              // 60
    • 除算
      •             
              12 / 5
              // 2.4
    • 剰余
      •             
              12 % 5
              // 2
    • インクリメント
      • x++
        •               
                var x = 3;
                x++; // ここでは x = 3
                x;    // この段階で x = 4 になる
      • ++x
        •               
                var x = 3;
                ++x; // ここで既に x = 4
    • デクリメント
      • x--
        •               
                var x = 3;
                x--; // ここでは x = 3
                x;   // この段階で x = 2 になる
      • --x
        •               
                var x = 3;
                --x; // この段階で x = 2 になる
    • 単項符号反転

      • 被演算子の符号を反転してその値を返す

                  
              var x = 3;
              -x;

              // -3が返ってくる。
    • 単項プラス

      • 数値でない被演算子の数値への変換を試みる

                  
              +"3";  // 3 (数値型)
              +true  // 1
              +false // 0
    • べき乗演算子
      •             
              2 ** 3;  // 2の3乗 = 8

              10 ** -1 // 10の-1乗 = 0.1
  • 論理演算子

    • 基本的にブール値（Boolean型：true / false）を用いる

    • 論理積 （AND：&&）
      •             
              true && true  // trueを返す
              true && false // falseを返す
              false && true // falseを返す

              false && (3 == 4) // falseを返す

              'Cat' && 'Dog' // "Dog"を返す
              false && 'Cat' // falseを返す
              'Cat' && false // falseを返す
    • 論理和 （OR ：||）
      •             
              true || true   // trueを返す
              false || true  // trueを返す
              true || false  // trueを返す
              false || false // falseを返す

              false || (3 == 4) // falseを返す
              false || (3 == 3) // trueを返す

              'Cat' || 'Dog' // "Cat"を返す
              false || 'Cat' // "Cat"を返す
              'Cat' || false // "Cat"を返す
    • 論理否定（NOT：!）
      •             
              !true  // falseを返す
              !false // trueを返す
              !'Cat' // falseを返す
