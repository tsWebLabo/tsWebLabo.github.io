配列

  • 配列とは
    • リストのようなオブジェクトである
    • 単一オブジェクト内に複数の値をリストとして保持している
    • 配列オブジェクトは変数に格納される
    • リスト内の値に個別にアクセスすることができる
  • 配列をつくる
    • 配列を定義してから要素を代入する方法
      •               
                var a = new array(); // 配列aを定義
                a[0] = 'dog';
                a[1] = 'cat';
                a[3] = 'hen';

                a.length; // 3
    • 配列の定義と要素の代入を同時に行う方法
      •               
                var b = ['dog', 'cat', 'hen'];

                b.length; // 3
  • 配列を呼び出す

    • 配列を呼び出すにはfor文を使って呼び出すことができます。

              
            for(let i = 0; i < a.length; i++) {
              console.log(a[i]);
            }

            "dog"
            "cat"
            "hen"
  • 項目の追加と削除
    •           
            // あらかじめ配列を定義しておきます
            let myArray = ['札幌', '仙台', '東京', '名古屋', '大阪', '博多'];
          
        
              
            // 配列の呼び出し
            for(let i = 0; i < myArray.length; i++) {
              console.log(myArray[i]);
            }

            札幌
            仙台
            東京
            名古屋
            大阪
            博多
    • 項目の追加

      • 配列の最後に項目を追加するにはpush()を使います。
      push()を使う時はひとつ以上の項目を引数に指定します。 

                  
              myArray.push('横浜');

              myArray.push('神戸', '広島', '京都');
            
          
                  
              // 配列の呼び出し
              for(let i = 0; i < myArray.length; i++) {
                console.log(myArray[i]);
              }

              仙台
              東京
              名古屋
              大阪

              横浜
              神戸
              広島
              京都
    • 項目の削除

      • 配列の最後の項目を削除するにはpop()を使います。 

                  
              myArray.pop();
