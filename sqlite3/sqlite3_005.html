SQLite3

SQLite3は、サーバーを用意する必要なく気軽に使えるデータベースです。
SQLite3を通してデータベースとSQL文に触れてみませんか？

既にテーブルに格納されているデータを更新したい時はUPDATE文を使います。
WHERE句をうまく使って効率よくUPDATE文を使えるようになりましょう。

データの更新（UPDATE文）

テーブルに格納されているデータを新しい値に更新するにはUPDATE文を使います。

書式

        
          UPDATE テーブル名 SET カラム名1 = 値1, カラム2 = 値2, ... WHERE 条件式;

まず更新の対象となるデータをWHERE句の条件式を使って指定します。
条件式に一致するデータが複数の場合、複数のデータがそれぞれ更新されます。
※WHERE句を省略した場合、テーブルに含まれる全てのデータが更新されます

次のテーブルを例にUPDATE文がどのようなものか見ていきます。 

        
          CREATE TABLE staff (id INTEGER, name TEXT, unit TEXT, flag TEXT);
          INSERT INTO staff VALUES (1, 'Yamada', 'Sales', 'Yes');
          INSERT INTO staff VALUES (2, 'Honda', 'Office', 'No');
          INSERT INTO staff VALUES (3, 'Nakai', 'Office', 'Yes');
          INSERT INTO staff VALUES (4, 'Suzuki', 'Tech', 'No');
          INSERT INTO staff VALUES (5, 'Itou', 'Sales', 'No');
        
      
        
          SELECT * FROM staff;

          id          name        unit        flag      
          ----------  ----------  ----------  ----------
          1           Yamada      Sales       Yes       
          2           Honda       Office      No        
          3           Nakai       Office      Yes       
          4           Suzuki      Tech        Yes       
          5           Itou        Sales       No

ひとつのデータを更新

idカラムの値が3のデータのnameカラムの値を変更

        
          UPDATE staff SET name = 'Nakajima' WHERE id = 3;
        
      
        
          SELECT * FROM staff;

          id          name        unit        flag      
          ----------  ----------  ----------  ----------
          1           Yamada      Sales       Yes       
          2           Honda       Office      No        
          3           Nakajima    Office      Yes       
          4           Suzuki      Tech        Yes       
          5           Itou        Sales       No

複数のデータを一度に更新する場合

unitカラムの'Offce'という値を'Desk'に変更します。

        
          UPDATE staff SET unit = 'Desk' WHERE unit = 'Office';
        
      
        
          SELECT * FROM staff;
          
          id          name        unit        flag      
          ----------  ----------  ----------  ----------
          1           Yamada      Sales       Yes       
          2           Honda       Desk        No        
          3           Nakajima    Desk        Yes       
          4           Suzuki      Tech        Yes       
          5           Itou        Sales       No

全てのデータを一度に更新する場合

flagカラムの値を'unknown'に変更してみます。

        
          UPDATE staff SET flag = 'unknown';
        
      
        
          SELECT * FROM staff;

          id          name        unit        flag      
          ----------  ----------  ----------  ----------
          1           Yamada      Sales       unKnown   
          2           Honda       Desk        unKnown   
          3           Nakajima    Desk        unKnown   
          4           Suzuki      Tech        unKnown   
          5           Itou        Sales       unKnown
