SQLite3のインストールとコマンドラインツールの起動

・SQLite3のインストール

ターミナルで以下のコマンド入力することでSQLite3のインストールを行えます

          
            sudo apt install sqlite3

・コマンドラインツールの起動

コマンドラインツールを使うことで、SQLite3を直接、対話的に使うことができます。

          
            sqlite3 sample.db

実行結果

          
            SQLite version 3.22.0 2018-01-22 18:45:57
            Enter ".help" for usage hints.
            sqlite>
