Express-generator

Express-generatorは、Webアプリケーションのひな型を生成してくれるツールです。

Express-generatorとは

Node.jsでWebアプリケーションを作成するのにExpressというフレームワークで「骨組み」をつくるという話をしました。
Express-generatorは、Expressをもとに、Webアプリケーションの「ひな型」を作成するツールです。
これによりWebアプリケーション作成を更に快適に開発できるようになります。

UIを作るテンプレートエンジン

ビューエンジン（WebサイトでのHTML的存在）は、EJS、Hbs、Pug、Twig、Vash等のビューエンジンを使うことができます。
このサイトでは、Pugを使っていきます。

スタイルシートは、CSSをそのまま使うこともできますが、Less、Sass、Compass、Stylus等の
一般的なスタイルシートエンジンを使うこともできます。

Express-generator導入

Express-generatorのインストール

ターミナルで下記コマンドを入力し、Express-generatorをグローバル環境にインストールします。 ※グローバル環境：パソコンのシステムの中にインストールするもので、一度インストールすれば何度でも使えます。 

          
            sudo npm install express-generator -g
          
        
          
            // カレントディレクトリにインストールする方法
            npm install express-generator --save

PCのシステムにインストールするのはちょっと・・・という方は、こちらでも使えるみたいです。
※この方法も僕の環境で試してみましたが、現状Hello World!の表示までできていません。
この場合は新しいプロジェクトごとにExpress-generatorをインストールする必要が出てくると考えられます。 

          
            // Express-generatorのヘルプ表示
            express --help

ここまででExpress-generatorの準備が整いました。
早速つくっていきましょう！！

プロジェクト作成
          
            // Sassを使わない場合
            express myApp --view=pug

            // Sassを使う場合
            express myapp --view=pug --css=scss

            change directory:
              $ cd myApp

            install dependencies:
              $ npm install

            run the app:
              $ DEBUG=myapp:* npm start
ディレクトリの確認
          
            ls

            myApp

            cd myApp
            ls

            app.js bin package.json public routes views

このようにmyAppディレクトリの中にたくさんのディレクトリやファイルが生成されています。

依存関係パッケージのインストール
          
            // myAppディレクトリに移動してから行います。
            npm install

この作業は、Express-generatorを使うのに必要なパッケージをインストールするものです。
必ずcd myAppでmyAppディレクトリに移動してから行いましょう。

実行してみよう！！
          
            npm start

このコマンドを入力した後、ブラウザでlocalhost:3000と入力してアプリにアクセスしてみましょう。
次のような表示がされれば成功です。
