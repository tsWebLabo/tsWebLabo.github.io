Ubuntuってなに？

  • LinuxディストリビューションOSのひとつ
  • Ubuntuは入門者に優しく設計されており、LinuxOSの中でも人気度が高い
  • 「Ubuntu」の語源・・・南アフリカのズールー語で『他者への思いやり』を意味する

Ubuntuの魅力

  • オープンソースのOSなので無料でつかえる。
  • 毎年4月と10月に新しいバージョンがリリースされるため、セキュリティ的にも安心
  • Ubuntuの日本語Remixもリリースされているので安心して使える

ロードマップがしっかりしていてセキュリティも安心

リリース時期 サポート期間 おすすめポイント
長期サポート版(LTS) 偶数年4月 5年間の無償セキュリティサポート サポート期間が長いので、安心して使える
通常版 奇数年の4月と毎年10月 9ヶ月の無償セキュリティサポート 半年ごとのリリースでより新しい機能を試したい方にオススメ

Ubuntuとその派生OS

Ubuntuは軽量でデスクトップPCのOSとしても優秀ですが、Ubuntuベースの派生OSもユニークなものが揃っています。

OS 用途 特徴
Xbuntu デスクトップ
  • Ubuntuと比較して軽量で安定している
  • 日本語環境もしっかりしている
Ubuntu Server サーバー
  • 自宅にサーバーを設置する際の選択肢のひとつとしていいかも
  • CUIのみの画面構成で、最もストレージの必要量が少ない
Ubuntu Studio クリエイター向けデスクトップOS
  • 画像処理・音楽制作・3Dモデリング等のソフトがプリインストールされている
  • Ubuntuと比較して多くのストレージが必要

Linuxコマンドを使ってみよう

Linuxコマンドを使えるとUbuntuとより仲良くなれます。
ここでは、個人的によく使うコマンドをまとめていきます。

ディレクトリ（フォルダ）操作
コマンド 機能 使い方（例文）
ls ファイルやディレクトリの情報を表示させるコマンド 
                  
                    ls
                    Documents  Music     Templates
                    Downloads  Pictures  Videos
                    Desktop    Dropbox   Public    snap
pwd 現在のディレクトリの場所を表示させる 
                  
                    pwd
                    /home/xxxxxx(ユーザー名)/
cd ディレクトリを移動するためのコマンド 
                  
                    pwd
                    /home/xxxxxx(ユーザー名)/
                    cd Desktop
                    pwd
                    /home/xxxxxx(ユーザー名)/Desktop/
                    cd ~
                    pwd
                    /home/xxxxxx(ユーザー名)/
mkdir 新しいディレクトリを作成する 
                  
                    ls
                    Documents  Music     Templates
                    Downloads  Pictures  Videos
                    Desktop    Dropbox   Public    snap
                    mkdir Scripts
                    ls
                    Documents  Music     Templates
                    Downloads  Pictures  Scripts   Videos
                    Desktop    Dropbox   Public    snap
ファイル操作
コマンド 機能 使い方（例文）
touch ファイルを新規作成する 
                  
                    touch sample.txt
                    ls
                    sample.txt
cat ファイルの内容を表示させる 
                  
                    cat sample.txt
                    これはsample.txtのテキストです。
cp ファイルをコピーする
mv
  • ファイル名を変更する
  • ファイルを移動させる
システム操作
コマンド 機能 使い方（例文）
sudo コマンドを管理者権限で実行する場合に使う
reboot PCの再起動
shutdown PCの電源OFF
