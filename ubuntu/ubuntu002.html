ターミナルでUbuntuがより便利になる！！

Linuxコマンドを使ってみよう

Linuxコマンドを使えるとUbuntuとより仲良くなれます。
ここでは、個人的によく使うコマンドをまとめていきます。

ディレクトリ（フォルダ）操作
コマンド 機能 使い方（例文）
ls ファイルやディレクトリの情報を表示させるコマンド 
                
                  ls
                  Documents  Music     Templates
                  Downloads  Pictures  Videos
                  Desktop    Dropbox   Public    snap
pwd 現在のディレクトリの場所を表示させる 
                
                  pwd
                  /home/xxxxxx(ユーザー名)/
cd ディレクトリを移動するためのコマンド 
                
                  pwd
                  /home/xxxxxx(ユーザー名)/
                  cd Desktop
                  pwd
                  /home/xxxxxx(ユーザー名)/Desktop/
                  cd ~
                  pwd
                  /home/xxxxxx(ユーザー名)/
mkdir 新しいディレクトリを作成する 
                
                  ls
                  Documents  Music     Templates
                  Downloads  Pictures  Videos
                  Desktop    Dropbox   Public    snap
                  mkdir Scripts
                  ls
                  Documents  Music     Templates
                  Downloads  Pictures  Scripts   Videos
                  Desktop    Dropbox   Public    snap
ファイル操作
コマンド 機能 使い方（例文）
touch ファイルを新規作成する 
                
                  touch sample.txt
                  ls
                  sample.txt
cat ファイルの内容を表示させる 
                
                  cat sample.txt
                  これはsample.txtのテキストです。
cp ファイルをコピーする
mv
  • ファイル名を変更する
  • ファイルを移動させる
システム操作
コマンド 機能 使い方（例文）
sudo コマンドを管理者権限で実行する場合に使う
reboot PCの再起動
shutdown PCの電源OFF
